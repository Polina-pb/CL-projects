# Чат-бот - составитель упражнений по юридическому нидерландскому

## Функционал чат-бота
Чат-бот может выполнять 2 функции:
1. Удалять из текста судебного решения Верховного суда Нидерландов (его фрагмента), присланного пользователем в формате .txt, все глаголы (на выходе пользователь получает текст с пропусками глаголов и списком удаленных глаголов в начальной форме); 
2. Направлять пользователю по его запросу текст судебного решения Верховного суда Нидерландов в формате .txt определенной длины и уровня лексического разнообразия со списком его ключевых слов.

## Актуальность
Отсутствие в открытом доступе подобных упражнений по юридическому нидерландскому, в особенности программ, которые могли бы использоваться преподавателями для формирования раздаточных материалов.

## Порядок работы чат-бота
Подробное описание работы чат-бота и этапов формирования корпуса судебных решений Верховного суда Нидерландов для выполнения им функции № 2 см. в соответствующих тетрадках и презентации.
